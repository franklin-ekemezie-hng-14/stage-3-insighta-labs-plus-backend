# HNG Internship Program

## Overview

The HNG Internship is a remote, fast-paced engineering program based in Nigeria that trains developers through real-world, production-grade tasks.

Participants work on staged backend, frontend, and full-stack challenges that simulate real industry systems.

---

## Structure

The program is divided into multiple stages:
- Stage 0: Setup & onboarding
- Stage 1: Data persistence & API design
- Stage 2: Query engine (filtering, sorting, pagination, search)
- Stage 3: Secure multi-interface system (this project)

Each stage builds on the previous one.

---

## Evaluation Style

Participants are evaluated based on:
- System design quality
- Code correctness
- Scalability thinking
- Security implementation
- Ability to follow strict requirements
- Consistency across interfaces

---

## Key Philosophy

- Real-world engineering > toy projects
- Strict adherence to specifications
- Production-grade thinking is required
- Partial credit is heavily reduced for incorrect response formats

---

## Stage 3 Focus

Stage 3 evaluates:
- Authentication systems (OAuth + PKCE)
- Role-based access control
- Multi-client architecture (CLI + Web + API)
- Secure token handling
- System consistency across interfaces

---

## Submission Expectation

Participants must submit:
- Backend repository
- CLI repository
- Web portal repository
- Live backend URL
- Live web URL

---

## Important Rule

All systems must behave identically across:
- API
- CLI
- Web interface

Consistency is heavily weighted in grading.
